## ✨ Descripción del proyecto

Health Dashboard es una aplicación frontend que simula un sistema de monitoreo de pacientes en un entorno clínico.

Permite visualizar información como:
- estado de pacientes
- signos vitales
- métricas de salud

El objetivo principal del proyecto es practicar **diseño de interfaces, estructura frontend y organización de código** usando JavaScript puro.

---

## 🎯 Funcionalidades actuales

- 📊 Panel de métricas (pacientes totales, críticos y estables)
- 🟢🔴 Indicadores de estado de salud
- 📋 Tabla de pacientes simulados
- 📈 Espacio reservado para gráficos futuros
- 🎨 Diseño tipo dashboard médico moderno

---

## 🧱 Tecnologías utilizadas

- Vite
- JavaScript (Vanilla)
- HTML5
- CSS3 (con variables y sistema de diseño)

## 🧠 Lo que aprendí con este proyecto

- Diseño de layouts tipo dashboard
- Uso de CSS variables para crear un sistema de diseño
- Organización de carpetas pensando en escalabilidad
- Construcción de UI sin frameworks
